# Quick AI Evolution Reference

## One-Line Commands

Tell Claude any of these to start AI evolution:

```
"Run AI evolution"
"Evolve the code pool"
"Improve the worst algorithm"
"Run evolution cycle"
```

## What Happens Automatically

1. ✅ **Simulation runs** (10k frames, ~10-20 seconds)
2. ✅ **Worst algorithm identified** (lowest reproduction rate)
3. ✅ **Code improvements generated** (bug fixes, optimizations)
4. ✅ **Changes committed** to git branch
5. ✅ **Pushed to remote** ready for PR

## Optional Follow-ups

```
"Run it longer to validate"          → 50k frame validation
"Show me the results"                 → Analysis of improvements
"Evolve again" or "next iteration"    → Improve next worst algorithm
"Create a PR"                         → Generate pull request
```

## Continuous Evolution Loop

```
User: "Run AI evolution"
Claude: ✅ CircularHunter improved!

User: "Merge the PR and evolve again"
Claude: ✅ Next worst performer improved!

User: "Keep going"
Claude: ✅ Third algorithm improved!
```

Repeat until all algorithms are optimized! 🧬
